# aether

A canvas-native creative system. Generate and edit assets directly on the canvas; pin AI-driven actions as reusable capabilities; fan out one hero scene into linked multiformat variants.

Built with Claude Opus 4.7 for the _Built with Opus 4.7_ hackathon (April 2026).

**Status.** Under active overnight construction — see [`docs/PRD.md`](./docs/PRD.md) and the [issue tracker](https://github.com/erniesg/aether/issues).
